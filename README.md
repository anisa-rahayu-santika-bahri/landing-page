## Setup

### Option 1: Use GitHub Pages

1. Create a new repository called `landing-page`
2. Upload the `index.html` file to your repository
3. Go to Settings â†’ Pages
4. Select the main branch as your source

### Option 2: Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required! 

## Customization

### Change Colors
Edit the gradient in the CSS:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Update Text
Modify the headline and subtitle in the HTML:
```html
<h1>Coming Soon</h1>
<p class="subtitle">Your custom message here</p>
```

### Social Links
Update the social media links in the HTML to point to your profiles.
